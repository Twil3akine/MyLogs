## 問題文

整数$N$が与えられます。$N^2$を求めてください

## 入力

入力は以下の形式で標準入力から与えられる。

```text
N
```

## 制約

<ul>
	<li>$-10^{2 \times 10^{5}} \le N \le 10^{2 \times 10^{5}}$</li>
</ul>

## 出力

答えを出力してください。

## 入力例1

```text
32
```

## 出力例1

```text
1024
```

$(2^{5})^{2} = 2^{10}$です

---

## 入力例2

```text
-32
```

## 出力例2

```text
1024
```

$N$が負の数でも二乗すると正になります。
