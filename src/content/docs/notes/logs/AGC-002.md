```c
int main(void) {
    ll a, b;
    scanf("%lld%lld", &a, &b);

    if (0<a)
        printf("%s\n", "Positive");
    
    else if (a<=0 && 0<=b) {
        printf("%s\n", "Zero");
    }

    else if (b<0) {
        printf("%s\n", ((b-a)%2==0) ? "Negative":"Positive");
    }

    return 0;
}
```

#### 解法

$a \le b$ が保証されているので、$0,\;a,\;b$の大小関係は$3$パターンに分けることができる。

1. $0 \lt a \le b$ のとき、必ず正となる。
2. $a \le 0 \le b$ のとき、必ず$0$となる。
3. $a \lt b \lt 0$ のとき、$(b-a)\%2=0$ならば負となり、$=1$ならば正となる。
   以上で題意を満たすことができる。
