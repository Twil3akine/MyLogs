I've updated the site for the first time in about a year and a half.
There isn't a particular reason, but I decided to update it because I felt a simpler site might be better. I wonder what you all think about it.

The design of this site is based on my <Marker>design policy</Marker>.

By the way, I started creating a component library thinking it would be easier for me to build sites in the future and intended to submit it to Astro Themes, but I lost sight of the goal halfway through and it's currently on hold. If you're interested, please check out the link below.

<LinkCard href="https://ravin-theme.pages.dev" />

Personally, I think it's a good idea for those whose design style is somewhat established to try creating a component library.
The reason is that it makes things overwhelmingly easier. It significantly reduces the time spent worrying about things like what the `border-radius` was supposed to be or how the colors were handled for each theme every single time.

I'm not sure yet if this design is fully finished or if I'll keep making small adjustments, but I think it's mostly complete.
